## マネタイズ戦略（2026/03/30 検討結果）

### 決定事項
- **無料 = 事実**: 今日どの魚が釣れたか、釣果一覧表示
- **有料 = 分析＋予測**: なぜ釣れたか（海況相関）、特定日の釣果予測
- **月額サブスク 380円/月**（都度払いは不採用）
- ターゲット: テクノロジーに疎い・一度購読したら辞めない・缶ジュース感覚で課金できる釣り人

### 重要な結論
現状の予測精度では「匹数予測」を売るのは早い。売るべきは「判断の方向性」:
- ✅ 方向性（好調/不調）、トレンド（先週比）、理由（海況相関）、エリア推奨
- ❌ 具体的な匹数予測（MAPE 113%では信頼性不足）

予測の見せ方: 匹数 → ★5段階評価＋トレンド＋理由テキストに変更する方針
