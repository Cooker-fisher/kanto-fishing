## 分析フレームワーク（全魚種共通）

### Step 1: データ集計
- CSV(data/*.csv)から魚種×エリアグループで抽出
- ポイント別（point_place）の件数・平均・最高・サイズを集計
- ポイント×月のヒートマップを作成

### Step 2: ベースライン構築
- **エリア×月平均**がベースライン（MAPE 41.6%@東京湾アジ）
- これが季節性を吸収し、最も安定した予測の土台になる
- ポイント別に分けるとさらに-12pt改善

### Step 3: 海況の影響分析
- **全体平均での比較は交絡に注意**（水温の季節変動など）
- 正しくは**エリア×月セグメント内**でlow/high群を比較
- 東京湾アジではエリア×月で切ると海況効果が消えた
- → 海況補正は匹数精度に寄与しない。コメント生成用に使う

### Step 4: 特徴量の個別検証
各特徴量を「ベースラインに追加して改善するか」でテスト:

| 特徴量 | 全体での見かけの効果 | セグメント内の実際の効果 | 予測への寄与 |
|--------|---------------------|------------------------|------------|
| ポイント別 | - | -12.8pt | **◎ 使う** |
| 水温偏差 | 冬に冷水で+36匹 | 方向バラバラ(平均差±0%) | × 交絡 |
| 波高 | 浅場で-32匹 | 凪27 vs 波高28（一貫しない） | × コメント用 |
| 風速 | 強風で-68匹@江戸川 | エリアで方向不一致 | × コメント用 |
| 風向 | 北西風で全体的に好調 | 未検証（使えるかも） | △ 要追加検証 |
| 波周期 | 短周期で+12匹 | 未確認 | △ |
| 月齢/潮汐 | ほぼ差なし | - | × 不要 |
| 台風 | 直撃で-65% | n数不足(1〜30件/条件) | × コメント用 |

### 時間粒度の検証結果
| モデル | MAPE | n数/セグメント | 除外件数 |
|--------|------|--------------|---------|
| **エリア×月** | **46.2%** | 平均33件 | 12件 |
| エリア×2週 | 46.2% | ほぼ同等 | 71件 |
| エリア×週 | 47.5% | 平均8件 | 454件 |

→ 2年分のデータでは**月単位が最適**。週にするとn数不足で悪化。
→ データ蓄積が3〜4年になれば週単位への切り替えを再検討。

### Step 5: バックテスト
- Leave-one-out方式（同エリア×月の他年月データで予測）
- 指標: MAPE, 中央値誤差, 20%/30%以内率
- **補正追加で悪化したら過補正** → capで制限 or 採用しない
- 乗算方式はノイズ増幅。加算方式+capが安定

### Step 6: 見せ方の決定
- **匹数は精度が限界 → レンジ表示（±1σ）**
- **型は高精度（MAPE 8.9%）→ 平均表示**
- **海況・台風 → 定性コメントで信頼感を出す**
- ★評価はエリア×月の相対スコアで1〜5

## 出船可否判断（共通ルール）

| 条件 | 欠航率 | 判定 |
|------|--------|------|
| 波高<0.5m & 風速<5m/s | 0% | 確実に出る |
| 波高<1.0m | 1-9% | ほぼOK |
| 波高1.0〜1.5m | 46% | 半分欠航 |
| 波高1.5m+ | 50%+ | ほぼ欠航 |
| 風速7〜10m/s | 24% | 欠航リスクあり |
| 風速10m/s+ | 52% | 半分欠航 |
| 波高1m+ & 風速5m/s+ | 46% | 半々 |

→ **波高1.0m or 風速7m/sが欠航リスクのライン**

## 台風の影響（コメント用）

- 直撃(<200km): 当日〜+1日は大幅減。+3日以降回復
- 遠方通過(300km+): むしろ好調になるケースあり
- n数不足で匹数補正には使えない
- コメント例: 「台風通過2日後。海況回復すれば好釣果の可能性」

### 庄治郎丸の問題点
1. ポイントデータ完全欠損（553件中0件）→ セグメント化不可

3. 匹数が低い（27%が10匹以下）→ MAPE分母が小さく膨れる
4. 個人差極端（同日2〜100匹の幅）

### 港別の海況感度
- 平塚港: 海況ほぼ無関係
- 大磯港: 海況ほぼ無関係。大潮がやや良い
- 小田原早川港: 風速に超敏感(+20.5匹)、長周期うねりで+14.5匹
- 松輪江奈港: 大潮で+11.7匹（相模湾で唯一潮汐が効く）

| 港 | MAPE(ベース) | MAPE(最良) | 改善手法 |
|---|-------------|-----------|---------|
| 富津港 | 42.5% | 41.7% | 船宿×ポイント×月+トレンド |
| 浦安 | 61.1% | 53.8% | 同上 |
| 長浦 | 74.4% | 65.6% | 水温補正+6.4pt |
| 鴨居大室港 | 70.9% | 66.6% | 船宿×ポイント×月+トレンド |
| 日立久慈港 | 59.5% | 69.9% | トレンドで悪化（回遊パターン不安定） |
| 平塚港(庄治郎丸) | 140.8% | 72.7% | **2隻分離で半減** |

### アジとの違い（海況の効き方）
| 要素 | アジ | タチウオ |
|------|------|---------|
| 水温 | ×交絡 | ★★★効く（夏:暖水+、冬:冷水+で逆転） |
| 潮差 | ±差なし | ★★小潮有利（中潮20.7匹 > 大潮16.5匹） |
| 月齢 | ±差なし | ★満月+4匹（70%一貫） |
| 波高 | 凪が良い | ★荒天で好調（濁りで活性UP？） |
| 風速 | ±差なし | ★弱い相関（6m/s超で-3匹） |
| 直近トレンド | ±効かない | ★★★-8pt改善（回遊魚だから「今来てるか」が重要） |

### 最良モデル
- 船宿×ポイント×月 + 直近7件トレンド(50/50): MAPE 68.1%（ベース76.1%から-8pt）

### 庄治郎丸の2隻分離
- 原因: 2隻体制で別々の釣果が同日2行で記録されている
- 混在MAPE 140.8% → 好調時のみ72.7%（半減）
- アジも同様: 良い船51.1%（混在66%から-15pt）
- 見せ方: 「好調時/控えめ時」の2パターン予測
  ```
  ⬆️ 好調時: 8〜19匹
  ⬇️ 控えめ時: 5〜14匹
  ※ 日によって差が出やすい船宿です
  ```
