## 東京湾アジ分析結果

### データ規模
- 過去2年で3,777件（海況付き3,754件）
- 主要ポイント: 木更津沖(610件), 横浜沖(549), 観音崎沖(408), 久里浜沖(346)

### ポイント別特性
| ポイント | 平均匹数 | 最高 | 型(cm) | 特徴 |
|---------|---------|------|--------|------|
| 木更津沖 | 59匹 | 713 | 25-28 | 数釣りの王道。冬がピーク(12月75匹) |
| 久里浜沖 | 52匹 | 242 | 34-39 | 数・型バランス◎。4月71匹が最強月 |
| 観音崎沖 | 34匹 | 160 | 36-40 | 型狙い最有力。10月40cm |
| 横浜沖   | 45匹 | 172 | 26-28 | 安定。冬が強い |
| 川崎沖   | 51匹 | 190 | 26-32 | 穴場。件数少ないが平均高い |

### 季節予測パターン
- 4月: 久里浜沖が41→71匹に急騰、富岡沖も37→61匹
- 木更津沖は4月以降下がる（56→53→41匹）
- 型は観音崎沖で夏〜秋に大型化（39-40cm）

### 海況感度（エリア別）
- 浦安: 波に超敏感（凪86匹 vs 波高55匹、差+32）
- 江戸川放水路: 風で壊滅（弱風84匹 vs 強風16匹）
- 久比里港: 波にも風にも強い（差±2匹）
- 横浜本牧: 風に敏感（弱風102匹 vs 強風77匹）
- 水温: アジは低水温の方が活性高い傾向（特に冬・秋）

### バックテスト結果
| モデル | MAPE | 30%以内 |
|--------|------|---------|
| 月平均のみ | 66.3% | - |
| +ポイント別 | 53.5% | 48% |
| +トレンド補正 | 50.3% | - |
| +海況補正(エリア別) | 44.6% | 53% |
| サイズ予測 | 8.9% | - |

### 見せ方の候補
- 候補1: 魚種×エリアの週次推奨（今のcalc_combo_scores拡張）
- 候補2: 特定日×ポイント予測（匹数レンジ + 型 + ★評価 + 海況理由）
  → 「4/5(土) 久里浜沖: 45〜65匹 / 35cm級 ★★★★☆ 凪予報で好条件」
