# 90_決定ログ

議論で決まったことを時系列で記録する。責任者が管理。

---

## フォーマット

```
### YYYY/MM/DD — 議題
- **決定**: 何を決めたか
- **理由**: なぜそう決めたか
- **関連ロール**: 影響するロール
```

## 既存の確定方針（CLAUDE.mdから転記）

### 2026/04/04 — マネタイズ方針確定
- **決定**: 無料=事実、有料=分析+予測。月額500円/スポット100円
- **理由**: 現状MAPE 27〜55%で匹数絶対値は売れない。「判断の方向性」を売る
- **関連ロール**: 全ロール

### 2026/04/04 — 予測の出力形式確定
- **決定**: 平年比±% + ★1〜5段階評価。例「来週末アジは平年比+25%（★★★★）」
- **理由**: 匹数絶対値ではなく方向性の提示が現実的
- **関連ロール**: デザイナー、プログラマー
