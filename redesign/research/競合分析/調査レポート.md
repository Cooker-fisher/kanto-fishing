# 競合・市場調査レポート（2026/04/04）

funatsuri-yoso.com のリデザインに向けた競合・市場調査の統合レポート。
6つの調査ストリーム（釣り情報サイト競合、予想課金サイト、課金心理、デザイントレンド、差別化戦略、信頼構築）の知見を統合する。

## 1. 釣り情報サイトの競合分析

### 1-1. 釣りビジョン (fishing-v.jp)

| 項目 | 内容 |
|------|------|
| 概要 | 釣り専門TVチャンネルのWeb版。年間約10万件の釣果データを保有 |
| データ形態 | 船宿個別ページのみ。横断集計なし、予測なし |
| デザイン | テーブル中心の古い日本的Webデザイン。レスポンシブ未対応（sp.サブドメインで分離） |
| **強み** | データ量の圧倒的な規模、TV局ブランドによる信頼性 |
| **弱み** | 横断比較が不可能、トレンド表示なし、予測機能なし |

当サイトのデータソースでもあるが、ユーザー体験としては「1船宿ずつ見に行く」しかできず、複数船宿を比較して意思決定する導線が存在しない。

### 1-2. 船釣り.jp (funaduri.jp)

| 項目 | 内容 |
|------|------|
| 概要 | 釣果横断集計＋年間トレンドグラフを提供。口コミ（気配り/設備/釣果・安全の3軸評価）あり |
| トレンドグラフ | 白=匹数、青=サイズ、ピンク=出船数の3線を年間で表示 |
| 技術基盤 | CGIベース |
| 価格 | 無料。関東限定 |
| **強み** | 横断集計と口コミの組み合わせ。無料 |
| **弱み** | 予測機能なし。CGIベースで技術的に古い |

**当サイトの最も直接的な競合**。事実の横断集計という点では同じ土俵にいるが、予測機能は持っていない。逆に言えば、事実の横断集計だけでは船釣り.jpとの差別化が困難。

### 1-3. 釣割 (chowari.jp)

| 項目 | 内容 |
|------|------|
| 概要 | 日本最大の釣り船予約サイト。3,000プラン、250万件の釣果データ |
| 独自機能 | 「爆釣タイム」: 潮汐ベースで時間帯別の釣れ具合を★表示 |
| ビジネスモデル | 予約→分析→レビューの一気通貫プラットフォーム |
| **強み** | データ量（250万件）、予約との連携による高い利便性 |
| **弱み** | 予約が中心で、予測は補助的な位置づけ。釣果分析の深さは限定的 |

「爆釣タイム」は潮汐のみに基づく簡易的な予測であり、気象・海況データを組み合わせた多変量予測とは精度・情報量が異なる。

### 1-4. ANGLERS (anglers.jp)

| 項目 | 内容 |
|------|------|
| 概要 | 日本最大の釣りSNS。650万件のユーザー投稿釣果、150万DL |
| 目玉機能 | ヒートマップ（「今どこで釣れてるか」密度マップ） |
| 有料プラン | サポーター月500円（広告非表示＋ヒートマップ過去30日＋限定アイコン） |
| **強み** | ヒートマップのビジュアル訴求力。SNSとしてのユーザーベース |
| **弱み** | ユーザー投稿データの品質にばらつき。船釣り特化ではない |

サポーター月500円は当サイトと同価格帯。ANGLERSはビジュアル（ヒートマップ）で有料の価値を訴求しており、参考になるアプローチ。

## 2. 予想課金サイトの分析（競馬・競艇・パチンコ）

釣果予測の課金モデルを設計するにあたり、「予想を売る」ビジネスとして成熟している競馬・競艇・パチンコ業界のサイトを分析した。

### 2-1. 競馬予想 (netkeiba.com等)

- **マーク体系**: ◎○▲△★の業界標準記号。瞬時に確信度が伝わる
- **信頼の核心**: 予想家ごとの的中率・回収率を公開。数字で信頼を裏付ける
- **無料/有料の切り分け**: 無料で過去結果と一部予想を公開 → 有料で全予想＋分析テキスト
- **ペイウォール手法**: ブラー（ぼかし）、ログイン壁、部分プレビュー、時間差公開の4パターン
- **価格帯**: netkeiba 500〜800円/月、予想家個別 300〜1,000円/回

### 2-2. 競艇予想（AI予想サイト等）

- **AI信頼度表示**: 小数点付きパーセンテージ（87.3%等）で科学的印象を演出
- **要因分解**: 水温/風/モーター性能等、何がスコアに影響したかを明示
- **処理アニメーション**: 「分析中...」スピナーで「計算している感」を演出
- **実績ページ**: 最重要の信頼シグナル。過去予想の的中/不的中を日付付きで公開

### 2-3. パチンコ攻略 (パチマガスロマガ等)

- **価格**: 月550円で機種スペック＋期待値表
- **信号機方式**: 緑/黄/赤で「打つ/様子見/避ける」を瞬時に伝達
- **信頼の源泉**: 雑誌ブランドの権威性がベース

### 2-4. 全予想サイト共通: 信頼構築の5要素

予想を売るサイトに共通する、ユーザーが課金に至るまでの信頼構築パターン:

| # | 要素 | 説明 |
|---|------|------|
| 1 | **的中実績の透明性** | 日付付きの過去予想vs結果ログ。的中も不的中も隠さず公開する |
| 2 | **★評価 + 理由の明示** | 星だけでは信頼されない。「なぜその評価なのか」の理由が必須 |
| 3 | **チラ見せ** | ブラー/部分表示/時間差で有料コンテンツの価値を体感させる |
| 4 | **運営者の顔** | 特定商取引法表記＋代表挨拶＋専門性の提示 |
| 5 | **無料実績の蓄積** | 2〜4週間無料で見て「当たる」ことを確認してから課金する導線 |

## 3. 釣り人の課金心理

### 3-1. 船釣りの支出規模

| 項目 | 金額 |
|------|------|
| 船釣り1回あたり | 13,000〜23,500円（乗船料＋交通費＋餌＋食費） |
| 月2回ペースの場合 | 月3〜5万円 |
| 年間 | 36〜60万円 |
| **500円/月の対乗船料比率** | **乗船1回の3〜5%** |

月500円は心理的ハードルが極めて低い。「1回の乗船で使う金額の数%で、その乗船の成果を最大化できる」というフレーミングが有効。

### 3-2. 釣り人が既に課金しているサービス

| サービス | 月額 | 提供価値 |
|---------|------|---------|
| 釣りビジョンVOD | 1,320円 | 釣り番組の動画視聴 |
| ANGLERS サポーター | 500円 | 広告非表示＋ヒートマップ拡張 |
| 釣りナビくん | 330円 | 釣り場情報 |
| Windy Premium | 166円/月 | 高精度気象データ |
| タイドグラフBI | 120円 | 潮汐データ |

500円/月は既存の釣り関連サブスクと同等の価格帯に位置する。

### 3-3. 課金トリガー TOP3

1. **「ハズレの日を避けたい」** -- 1万円以上使ってボウズ（釣果ゼロ）は精神的にも経済的にも痛い。「行かない判断」に価値がある
2. **「いつ行けば釣れるか知りたい」** -- 限られた週末のうち、最適な日を選びたい
3. **「何を狙うか迷っている」** -- 今週ベストな魚種＋船宿の組み合わせを知りたい

共通するのは**「限られたリソース（時間・金）の配分を最適化したい」**という動機。

### 3-4. Aha moment（無料→有料の転換点）

ユーザーが課金を決断する瞬間:

- チラ見せ予測が的中した体験（「あの予想、当たってた」）
- ハズレ日を見送って正解だった実感（「行かなくてよかった」）
- 無料の事実データと有料の予測の**「差分の大きさ」**を体感した瞬間

## 4. デザイントレンド分析

### 4-1. 成功パターン

| パターン | 出典 | 内容 |
|---------|------|------|
| **カード別ビジュアル処理** | Apple Weather | カードごとに異なるビジュアル（温度はグラデーション、UVはアーク、降水はバーチャート）。「同じ白カードの壁」を避ける |
| **色グラデーション** | Windy | 紫→青→緑→黄→赤でデータを直感的に表現 |
| **写真+グラデーション+ブラー** | Fishbrain | 魚種写真＋グラデーション＋ヒートマップ。有料コンテンツはソフトブラーで隠す |
| **Bento Grid** | Apple等 | カードサイズに差をつける。TOP1は2倍サイズ。自動的に視覚階層ができる |
| **信号機方式** | パチンコ攻略サイト等 | 緑/黄/赤で瞬時に判断可能 |

### 4-2. 避けるべきパターン

- **全カード同サイズ同形状**: 視覚階層がなく、どれが重要か分からない（当サイト現状の問題）
- **テキストだけのヒーロー**: 視覚的インパクトが弱い
- **データベース感**: テーブル中心のレイアウト（釣りビジョンの問題）
- **競合と同じ色使い**: 白背景に青アクセントはどこにでもある。差別化にならない

## 5. 結論: funatsuri-yoso.com の差別化戦略

### 5-1. 唯一の有料化可能な独自価値 = 「予測」

既存サービスとの機能重複を整理すると、有料化できる独自領域は明確に1つ:

| 機能 | 既存の無料/安価な代替 | 独自性 |
|------|---------------------|--------|
| 事実の横断集計 | 船釣り.jp（無料） | なし |
| 潮汐情報 | 釣割/タイドグラフBI（120円） | なし |
| 海況情報 | Windy（166円/月） | なし |
| **釣果予測（平年比+★評価）** | **なし** | **あり** |

> **「来週末のアジは平年比+25%（★★★★）」は誰も出していない。**

純粋な釣果予測サブスクリプションの大成功例はまだ存在しない。これは**先行者利益**を取れる領域。

### 5-2. 実装すべき信頼構築要素

予想課金サイトの共通知見から、以下の5つを実装に組み込む:

1. **実績ページ**: 過去予想vs実績の日付付きログ（的中も不的中も公開）
2. **的中率の定義を明示**: 「方向（上/下）が当たれば的中」等、基準を透明化
3. **★評価 + 理由テキスト**: 「水温上昇中＋大潮＋南西風弱」のように根拠を示す
4. **運営者情報＋特定商取引法表記**: 法的要件かつ信頼シグナル
5. **無料→有料の転換ファネル**: 2〜4週間の無料検証期間を設計に組み込む

### 5-3. デザインで差別化すべき点

- **ダーク＋ゴールドの「プレミアム感」**: 競馬予想サイトのような高級感。白背景の競合との差別化
- **魚種ごとの色分け**: アジ=青、マダイ=赤、タチウオ=銀 等
- **Bentoグリッド**: TOP1大カード＋残り小カードで視覚階層を作る
- **グラデーションフェード型ペイウォール**: 有料コンテンツの存在を示しつつ、中身はぼかす
- **信号機式の出船判定**: ◎○△× で「行くべきか」を瞬時に伝える
- **スパークライン**: 7日間トレンドのミニチャートで変化の方向を直感的に表示
