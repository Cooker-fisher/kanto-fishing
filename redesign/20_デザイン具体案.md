## 2. 配色テーマ

### 方針: ライトテーマに変更（2026/04/04確定）

現行ダークネイビーからライトテーマに刷新。白背景ベースで清潔感・信頼感を出す。
釣り情報サイトとして「海の青」をアクセントに、**8色のコアパレット**でCSS変数管理。

### 新パレット（ライトテーマ）

```css
:root {
  /* ── ベース ── */
  --bg-primary:    #f5f7fa;   /* メイン背景（薄いグレー） */
  --bg-card:       #ffffff;   /* カード・セクション背景（白） */
  --bg-input:      #eef1f5;   /* 入力・ナビ背景 */
  --border:        #d0d8e0;   /* 汎用ボーダー */

  /* ── テキスト ── */
  --text-primary:  #1a2332;   /* メインテキスト（濃紺） */
  --text-secondary:#5a6a7a;   /* サブテキスト */
  --text-muted:    #8a96a4;   /* 注記・補足 */

  /* ── アクセント ── */
  --accent:        #0077cc;   /* 主アクセント（海の青・リンク・見出し） */
  --accent-hover:  #005fa3;   /* ホバー時（やや暗く） */
  --cta:           #e85d04;   /* CTA・スコア・重要アクション（橙） */
  --cta-hover:     #d04e00;   /* CTAホバー */

  /* ── セマンティック ── */
  --positive:      #1a9d56;   /* 好調・UP（緑） */
  --negative:      #d43333;   /* 不調・DOWN（赤） */
  --warning:       #d4a017;   /* ★評価・注意（黄） */
  --premium:       #7c3aed;   /* 有料コンテンツ識別（紫） */

  /* ── ナビ・ヘッダ ── */
  --header-bg:     #0d2b4a;   /* ヘッダ背景（濃紺・海のイメージ残す） */
  --header-text:   #ffffff;   /* ヘッダテキスト */
  --nav-bg:        #f0f3f7;   /* ナビ背景 */

  /* ── サイズ ── */
  --radius-sm:     6px;
  --radius-md:     10px;
  --radius-lg:     14px;
  --max-width:     1100px;
}
```

### 変更点
| 現行 | 新 | 理由 |
|------|---|------|
| ダークネイビー背景 | 白/薄グレー背景 | 視認性向上・一般的な情報サイトとして信頼感 |
| 15色がハードコード | 8色+ヘッダ系のCSS変数 | 一元管理・テーマ変更容易 |
| ヘッダもダーク | ヘッダのみ濃紺を残す | 海のイメージを保持しつつ、本文は明るく |
| アクセント青`#4db8ff` | `#0077cc`（濃い目の青） | ライト背景でのコントラスト確保 |

## 3. ナビゲーション設計

### 現状の問題
- ナビが1段（トップ / カレンダー / エリアドロップダウン）で、魚種への導線が弱い
- モバイルで `<nav>` が横スクロールに頼っている

### 新ナビ構造

```
┌─────────────────────────────────────────────┐
│ 🎣 船釣り予想        [検索🔍]  [≡ メニュー] │  ← モバイル
├─────────────────────────────────────────────┤
│ PC: 今日の釣果 | 魚種 | エリア | カレンダー | 予測(有料) │
└─────────────────────────────────────────────┘
```

#### モバイル（≤768px）
- **ヘッダ**: サイト名（左）+ 検索アイコン + ハンバーガーメニュー（右）
- **ドロワーメニュー**: 右からスライドイン。主要リンク + エリア一覧 + 魚種一覧
- **ボトムナビ（固定）**: 釣果 / 魚種 / エリア / 予測 の4アイコン

#### PC（>768px）
- **ヘッダ**: サイト名（左）+ グローバルナビ（中央）+ 検索バー（右）
- **ドロップダウン**: 「魚種」「エリア」にホバーで展開メニュー

#### SEO対応
- `<nav aria-label="メインナビゲーション">` でアクセシビリティ対応
- パンくずリスト（JSON-LD BreadcrumbList）: トップ > 魚種 > アジ

## 7. AdSense審査対応

### 新設ページ

| ページ | ファイル | 内容 |
|--------|---------|------|
| プライバシーポリシー | privacy.html | Cookie、GA、データ取扱い |
| 利用規約 | terms.html | 免責事項、著作権、予測の注意事項 |
| サイトについて | about.html | サイトの目的・運営者・データソース |

### SEO技術対応

| 対応 | 実装方法 |
|------|---------|
| sitemap.xml | build_sitemap() 既存 → fish_area/forecast ページも追加 |
| robots.txt | 新規作成（静的ファイル） |
| 構造化データ（JSON-LD） | WebSite + BreadcrumbList + FAQPage（魚種ページ） |
| OGPタグ | 全ページに og:title, og:description, og:image |
| canonical URL | 全ページに `<link rel="canonical">` |
| meta description | ページごとに動的生成（魚種名・件数・日付を含む） |

### コンテンツ品質改善

| 現状 | 改善 |
|------|------|
| 数値データのみ | 各魚種ページに「{魚種}とは」の解説テキスト追加 |
| 機械的な一覧 | 「今日のポイント」サマリー文を自動生成 |
| 釣りビジョンのデータそのまま | 独自の集計・分析視点を明示 |

## 8. CSS構造（プログラマーへの指示）

### 現状 → 新構造

```
現状: CSS 5箇所に分散（関数内ローカル変数）
      ↓
新:   style.css（外部ファイル1つ）+ ページ固有の差分はインライン最小限
```

### ファイル構成案

```
kanto-fishing/
├── style.css          ← 新規：共通CSS（CSS変数・リセット・レイアウト・コンポーネント）
├── main.js            ← 新規：共通JS（フィルタ・ソート・タブ切替・GA）
├── index.html         ← <link rel="stylesheet" href="style.css">
├── fish/*.html         ← 同上
├── area/*.html         ← 同上
└── ...
```

### crawler.py 変更方針

| 変更 | 内容 |
|------|------|
| `_page_head()` 新設 | 共通ヘッダ・meta・CSS読み込み・ナビを生成する関数 |
| `_page_foot()` 新設 | 共通フッタ・JS読み込みを生成する関数 |
| `_page_nav()` 新設 | グローバルナビ（パンくず対応）を生成する関数 |
| `CSS` 変数削除 | style.css に移行。crawler.py 内のCSS文字列を廃止 |
| `build_style_css()` 新設 | style.css ファイルを生成する関数 |
