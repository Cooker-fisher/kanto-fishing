# 11_有料ページデザイン

**管理者**: デザイナー

## 目的

有料コンテンツの設計。月額500円 / スポット100円の価値を感じてもらえるUI・コンテンツ設計。

---

## 方針（ユーザー/CLAUDE.md確定済み）

| 項目 | 決定内容 |
|------|---------|
| 有料の定義 | **分析＋予測**（なぜ釣れたか、来週どうか） |
| 価格 | 月額500円 / スポット100円 |
| 予測形式 | 平年比±% + ★5段階評価 |
| 出力例 | 「来週末アジは平年比+25%（★★★★）」 |
| ペイウォール | チラ見せ方式 |

## 予測の出力形式設計

CLAUDE.md確定：匹数絶対値ではなく「判断の方向性」を売る

```
┌──────────────────────────────────────┐
│ 4/5(土) マダイ                        │
│ ★★★★☆  平年比 +25%                 │
│                                      │
│ 理由: 大潮×SST上昇×波穏やか          │
│ 推奨エリア: 茨城（日立久慈港）         │
│ 注意: 午後から南風強まる予報           │
└──────────────────────────────────────┘
```

## ペイウォールUI設計

### チラ見せの設計原則
1. **何が見えるか**を明確に見せる（価値を感じさせる）
2. **あと一歩**で手が届く感覚（ストレスは最小限）
3. **スポット100円**のハードルを低く見せる

### UI案（デザイナーが具体化）
- [ ] ぼかし（CSS blur）+ 「有料会員限定」オーバーレイ
- [ ] 1行だけ見えて残りフェードアウト
- [ ] 鍵アイコン + CTA「100円で今すぐ見る」
- [ ] 無料登録（メアド）→ 一部閲覧可 → 有料で全解放
