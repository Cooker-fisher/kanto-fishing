## 概要

分析精度・データフロー・フォルダ構成の改善を担うロールシステム。

## ロール一覧

| ファイル | ロール | 責任 |
|---------|--------|------|
| 00_責任者.md | プロジェクトマネージャー | 方針管理・ロール間調整 |
| 01_上層担当.md | 上層データ担当 | CSVスキーマ・データ品質 |
| 02_下層担当.md | 下層伝達担当 | C層→E層データフロー設計 |
| 03_GitHub担当者.md | GitHub担当 | ブランチ管理・crawl.yml |

## このプロジェクトで管理する4テーマ

1. **テーマ1**: 分析結果の改善（MAPE削減・combo_deep_dive全種実行）
2. **テーマ2**: 分析方法の改善（手法選定・変数最適化）
3. **テーマ3**: 下層伝達設計（C層→E層データフロー）
4. **テーマ4**: 答え合わせ分析方法（予測vs実績検証フレームワーク）
