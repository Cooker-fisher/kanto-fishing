## チェックリスト

### Python構文

- [ ] `python -m py_compile <file>` が通るか
- [ ] 型アノテーションの矛盾がないか（Optional未使用でNoneを返す等）
- [ ] f文字列の変数名が存在するか
- [ ] インデントエラー（特にif/for/with のブロック）

### SQLクエリ

- [ ] WHERE句が意図通りか（AND/OR の優先順位）
- [ ] JOIN条件が正しいか（粒度の異なるテーブルをそのままjoinしていないか）
- [ ] GROUP BY が SELECT 列と一致しているか
- [ ] サブクエリ内の集計が正しいか
- [ ] `is_cancellation = 0` フィルタが抜けていないか

### 条件分岐

- [ ] off-by-one（`<` vs `<=`、`>=` の使い分け）
- [ ] None未チェック（`if x:` で 0 が偽になる問題）
- [ ] 型ミスマッチ（文字列 vs 数値の比較）
- [ ] 空リスト/空DataFrameのガードがあるか

### ループ・件数確認

- [ ] **ループ実行前に `print(f'対象: {len(list)}件 先頭3: {list[:3]}')` があるか**（CLAUDE.md必須ルール）
- [ ] 件数0の場合のガード処理があるか
- [ ] ネストループの計算量が現実的か

### ハードコード

- [ ] マジックナンバーがないか（定数として定義されているか）
- [ ] ファイルパスがハードコードされていないか（`_paths.py` 使用）

### エラーハンドリング

- [ ] SQLiteアクセス失敗時の対処があるか
- [ ] ファイル不在時の対処があるか
- [ ] `try/except Exception as e: pass` の無言無視がないか
