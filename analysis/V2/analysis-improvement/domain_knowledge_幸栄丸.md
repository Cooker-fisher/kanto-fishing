## 漁場構造の概要（鹿島港）

鹿島港は茨城県南東部・鹿島灘に面した外洋系漁港。東京湾・相模湾と異なり内湾の影響がなく、潮通しと開放性が高い。

| ポイント | 位置 | 水深目安 | 海底地形 |
|---------|------|---------|---------|
| 鹿島北沖 | 港から北東方向 | 30〜50m | 岩礁混じり砂地。根周りあり |
| 鹿島真沖 | 港から真東方向 | 60〜80m | 砂泥底。比較的平坦 |
| 鹿島南沖 | 港から南東方向 | 20〜40m | 砂礫帯。穏やかな海況が多い |
| 大根 | 沖合遠方 | 中〜深場 | 海底の岩礁・根。青物回遊帯 |
| 鹿島魚礁 | 人工魚礁付近 | 中程度 | 設置魚礁。根魚・底もの全般 |
| 鹿島湾内 | 防波堤内側 | 浅い | 荒天時避難・内湾向け魚種 |

**wave中央値の比較**（weather_cache.sqlite 実績）:
- 北沖: 1.23m / 真沖: 1.19m / 南沖: 0.75m

南沖は有意に穏やかな条件で選択される傾向がある。

## ドメインチェック結果（2026-04-22）

### 1. ヒラメ→鹿島北沖 70%（N=44）: 妥当

**判定: 妥当**

北沖の岩礁混じり砂地という地形は、ヒラメがベイト（イワシ・カタクチイワシ）を待ち伏せる環境として典型的。マハタ（68%、N=25）が同ポイントに集中するのも「ベイトが同エリアに集まる」という食物連鎖の共通要因で合理的。ヒラメ（待ち伏せ型）とマハタ（根着き型）という異なる戦略の魚が同ポイントに集まるのは、「岩礁周りのベイト密度が高い」という共通の引力による。

**要注意**: 気象推定ルール③の「wave≥1.5m + 東寄り風(NE/E/N) → 鹿島北沖」は**反直感的**。鹿島灘では北東〜東からの風波が最も危険であり、通常の現場感では南沖・湾内へ避難する。このルールは実欠航記録との突き合わせ検証が必要。

---

### 2. マダイ→鹿島南沖 67%（N=6）: 妥当（N薄い・要月別確認）

**判定: 釣り実務的には妥当。ただし N=6 は統計的に薄い**

- 南沖は水深20〜40mの砂礫帯でテンヤ船の定番漁場として合理的
- 乗っ込みマダイ（4〜6月）は産卵前に浅場へ集まる習性と一致
- wave中央値が南沖で最も低い（0.75m）→「テンヤ釣りは繊細→凪の日に南沖」という選択バイアスとも整合

**注意点**:
- N=6（実記録）は薄い。魚種デフォルト補完後のN=512と混同しないこと（補完後Nは②推定値が大半）
- 乗っ込み期（4〜6月）に絞った月別集計での裏付けを推奨する

---

### 3. 夏季+凪+南風→大根（青物）: 妥当

**判定: 妥当**

3条件すべてが釣り実務的に独立した理由で正しい：
- SST≥20℃（鹿島沖では概ね7〜9月）はカツオ・ワラサの索餌圏北上期と一致
- 南寄り風は黒潮系暖水の北上を促す気象条件と連動
- wave<0.8mの凪は「距離の遠い大根への出船安全マージン」として必須条件

SST中央値が大根22.9℃（北沖16.3℃・南沖18.6℃と比較して顕著に高い）という実績値が、「夏季高水温期のみ使われるポイント」という解釈を定量的に裏付けている。

**要注意**: 「大根」は海底の岩礁・根を指す固有地名。`point_coords.json` の座標定義の正確性が気象特徴量取得精度に直結する。カツオ・ワラサは KAIYU_FISH であり★チャンス評価精度に直接影響するため、座標確認を優先すること。

---

### 4. ヤリイカ→鹿島真沖 86%（水深60-70m固定）: 妥当

**判定: 妥当。ヤリイカの生態と完全に一致**

- ヤリイカは水深50〜100mの砂泥底底生型。60〜70mは適水深の中心値
- 他のイカ類と比べて「水温より底質・水深に依存し、シーズン中は同じ漁場を使い続ける」という低移動性の現場知見と合致
- 86%という高い集中度はこの漁場固定性を忠実に反映している

**追加示唆**: 冬（浅め）・春（深め）という季節的水深変動の可能性がある。月別水深分布を確認すると、真沖を月別に細分化できる可能性がある。

## ワラサの分散ポイントについて

| ポイント | 比率 | ドメイン解釈 |
|---------|------|------------|
| 北沖 | 38% | 冬〜春のイワシ群を追う時期 |
| 真沖 | 25% | 中間水深での待ち |
| 大根 | 19% | 夏〜秋の沖合回遊群 |
| 南沖 | 13% | 凪の日限定（少ない） |

4ポイントへの分散はワラサが回遊魚である事実と合致。気象推定ロジック（wave/SST/wind_dir複合条件）の方向性は正しい。ただしポイント別よりも「SST×月」の複合条件でのセグメント分けのほうが回遊実態に近い可能性がある。

## 気象→ポイント推定ルールの評価

| 推定ルール | ドメイン評価 | コメント |
|-----------|------------|---------|
| wave≥1.5m + 東寄り風(NE/E/N) → 鹿島北沖 | **要検証** | 鹿島灘では東〜北東風が最も危険。南沖・湾内への避難が現場的。このルールは反直感的 |
| wave≥1.5m + 西/南西系風 → 鹿島真沖 | 概ね妥当 | 西〜南西風は鹿島灘では比較的よくある。真沖への影響は中程度 |
| wave<0.8m + SST≥20℃ + 南寄り風 → 大根 | 妥当 | 凪・暖水・南風の3条件は夏季青物のシグナルとして現場感に合致 |
| wave 0.8-1.5m + NW/W → 鹿島北沖 | 概ね妥当 | 北西風は鹿島灘では陸側から吹く風。沖出し可能で北沖へ |
| wave<0.8m + NE/E → 鹿島南沖 | 要確認 | 東寄り凪は珍しい条件。南沖選択との関連は不明 |

**最優先で要確認**: `wave≥1.5m + 東寄り風 → 鹿島北沖` のルール。実際の出船・欠航記録と組み合わせて高波×東風時の北沖出船実績を確認すること。
