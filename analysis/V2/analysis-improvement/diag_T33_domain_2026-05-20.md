## A. 水色が「効く魚種」「効かない魚種」の現場感

### 水色が決定的に効く魚種（上位）

**1. マダイ（特に乗っ込み期・相模湾〜外房）**
- 適水色: 「やや澄み〜薄濁り」。黒潮縁辺の青みがかった海水と接する砂礫帯が産卵前の集積場になる
- 「真っ青」でも「泥濁り」でも食いが落ちる典型的な中間適正型
- 現場感: 相模湾のテンヤマダイは「潮色が悪い（濁り）と反応が消える」が定番の船長コメント
- 「黒潮の影響あり → 青み帯びた澄み → 釣れる」非線形パターンが存在

**2. シロギス（東京湾内湾・相模湾砂地帯）**
- 澄みでも釣れるが、「薄濁り（3）」が最良という現場感が圧倒的に強い
- 理由: キスは砂底の小生物を視覚で追う捕食者。透明すぎると活性が落ちる（逃げやすくなる）
- 雨後数日の軽い濁りで釣果が急増する典型魚種

**3. シーバス（東京湾・多摩川・江戸川放水路近傍）**
- 「濁り＝シーバス」は現場の絶対的常識。ルアー狩猟系で視認性が下がる＝奇襲が決まりやすい
- 多摩川・荒川河口周辺では春先の増水後濁りが大型出現トリガーになる
- tone_impact / tama_impact の river proximity 特徴量は既に water_color_model.py に実装済

**4. メバル（根周り・薄濁り選好）**
- 夜行性傾向が強く、昼間は薄暗い状況＝薄濁りで食いが立つ。東京湾内湾系で特に顕著

**5. マゴチ（浜名湖〜相模川河口周辺の砂地）**
- シロギスと同じ砂地系ベイト依存。「薄濁り〜やや濁り」が最良

**6. タチウオ（東京湾・内湾系）**
- 潮汐と水深依存が水色より大きいが、「青みが強い澄み＝外洋水の流入」があると良型が多い

### 水色が効かない・無関係に近い魚種

- **ヤリイカ・スルメイカ・マルイカ**: 水深50m以上の中深場で底生的に生活
- **キンメダイ・メダイ・アマダイ**: 水深200m〜の外洋深場。SLA/黒潮位置の方が有効
- **カツオ・キハダマグロ・シイラ**: 外洋回遊系。CHL/SST/SLA が決定因子
- **カワハギ**: 根周りの底もので光量・視認性への感受性が低い
- **オニカサゴ・マハタ**: 岩礁定着型で回遊行動が薄い

### 非線形パターン（注目）

水色が単調でなく「ある状態で最良、両極で悪化」する魚種:

- **マダイ**: 澄み(1)と濁り(5)の両端で釣果低下。薄濁り(3)付近がピーク
- **シロギス**: 澄み(1)で低下、やや濁り(4)超でも低下。薄濁り(3)が最良
- **ヒラメ（外洋向き）**: 黒潮系青み澄みでベイト溜まる → 連動釣果増

→ 現在 combo_deep_dive で water_color_n の **直線回帰のみ** だが、「水色 × 水色の二乗」を追加すると非線形が拾える

### 「水色予測精度向上で改善効果が大きい 魚種×エリア」

| 優先度 | 魚種 | エリア | 理由 |
|--------|------|--------|------|
| 最高 | シロギス | 相模湾・東京湾砂地帯 | 薄濁り依存が強く、雨後ラグが短期に影響 |
| 高 | シーバス | 東京湾（浦安・江戸川・平和島） | 河口濁りが即時影響。tama/tone impact 連携 |
| 高 | マダイ | 相模湾（長井・松輪・葉山）、外房（大原） | 中間適正型の非線形が掴めれば精度向上余地大 |
| 中 | メバル | 東京湾内湾全般 | 薄濁り選好が強いが water_color 非空率が低い |
| 中 | カマス | 外房（庄治郎丸コンボあり） | 外洋系だが湾口では水色変化に反応 |

## B. 水色情報がないエリアの代替補間の妥当性

### 関東圏で水色観測が「薄い」エリア

- **鹿島沖・茨城系**: kanso_raw 非空率が低い。外洋開放系で潮通しが良く水色変動幅が小さい
- **御前崎・福田・田子の浦（静岡西部）**: 遠州灘向き。外洋性が強く黒潮の直接影響を受ける
- **下田・龍正丸エリア（南伊豆）**: 水深急変・外洋性高い

### 隣接エリア補間の妥当性判断

**御前崎 ← 下田南沖の水色を引く: 「条件付き妥当」**
- 同じ黒潮接岸エリアで水色パターンが連動しやすい
- ただし御前崎は大井川・天竜川の河川影響がある内側（浅場）と、外洋側で完全に異なる
- 判断: **深場（50m超）コンボのみ補間を適用し、浅場は別扱い**

**鹿島南沖 ← 犬吠埼沖の水色を引く: 「要注意」**
- 鹿島南沖は利根川河川影響を受ける内側
- 南沖は近傍波崎港（tone 河口影響圏）の水色の方が代理として近い
- 判断: 鹿島系は「ポイント別に補間元を選ぶ」

**東京湾内湾 ← 外房の水色を引く: 「NG」（最重要）**
- 東京湾口（観音崎〜野島崎ライン）は水色の「壁」
- 内湾と外洋は別海域で年間を通じて常態的に異なる
- IDW 単純補間で外房 → 内湾に水色値が滲み出すと全コンボに系統的な誤差が入る
- 判断: **AREA_TO_ZONE をゾーン境界として補間を「ゾーン内のみ」に制限**

**相模湾内部の補間: 「概ね妥当」**
- 葉山〜茅ヶ崎〜平塚〜小田原は同じ相模湾で黒潮の縁辺影響を共有
- ただし真鶴半島付近の地形起因の潮流乱れに注意

**距離より「同ゾーンフラグ」補間への示唆**
- AREA_TO_ZONE が既に定義されている。このゾーン区分を補間の制限フィルターとして使う
- ゾーン内で最近傍 N 点の水色を平均、ゾーンをまたぐ場合はフォールバック（季節平均等）
- IDW はゾーン内なら有効。ゾーンまたぎの IDW は精度が下がるリスク

## C. 新規 chowari 系船宿の特性

### 地理的分布の傾向

- 庄治郎丸（外房・飯岡）、たいぞう丸（外房系）、弘漁丸、勇盛丸、勇幸丸（内房〜外房）
- 龍正丸（下田）、恵漁丸、増福丸（伊豆系）
- 太幸丸、村井丸（内房富浦・金谷）
- chowari は「関東近郊の家族・週末アングラー向けの中小船宿」が中心

### chowari 系の釣果データ特性

**point_place1 が薄い**: 中小船宿は「コメント欄に行き先を書かない」文化が多い。曖昧な記述か無記述が多く、point_coords.json の解決率が fishing-v 系より低い可能性

**季節限定運航**: 外房・内房系は特定魚種のシーズン中のみ集中出船。leave-one-month-out CV で特定月のデータが極端に薄くなるリスク

**ターゲット魚種が明確**: chowari 船宿は専門化率が高い（ヒラメ専門・アジ専門等）。combo の n は速く 30 件超えるが、魚種間の分散が極めて小さい

### 予測手法を分けるべきか

「分ける必要はないが、補正の優先順位が異なる」:

- **point_place1 欠損率が高い場合**: ship_fish_point.json の「船宿×魚種デフォルトポイント」フォールバックへの依存が高くなる。**デフォルトポイントの座標精度が予測精度の上限を規定** → chowari 系新規船宿については point 登録の整備が最優先の投資対象
- **季節偏りへの対処**: MIN_MONTHS=6 ガードが特定月ゼロのコンボを弾く可能性。外房の夏イサキ船宿はデータが6〜9月に集中するため月数が足りない問題が起きやすい。「季節集中コンボ専用の MIN_MONTHS 緩和（例:3ヶ月）」で対応可能
- **fishing-v 系との融合**: 同じ魚種×近傍エリアで両者が共存するコンボが増えてきた場合、エリア集計（area_decadal）レベルで統合ベースライン可能性

## D. 全体最適化の現場アイデア

### 「効くはず」の未実装シグナル 3 件

**1. 前日からの風向転換（南→北、東→西）**
- 「風が変わると潮目が動く」は関東船釣りの普遍的現場感
- 南風が長く続いた後に北風に変わると東京湾内湾では水が入れ替わり、マダイ・タチウオの食いが急に立つ
- 現在 wind_dir は当日の絶対方向のみ。差分（wind_dir_t - wind_dir_t-1）の cos/sin 変化量として特徴量化
- FAST 因子なので H<=7 で有効

**2. SST が連続 X 日下がり続けているフラグ（sst_downtrend_n）**
- 現在 sst_delta_3d（3日差分）は実装済みだが、「X 日連続低下中」という趨勢を表す特徴量がない
- 水温低下が 3 日続くとカワハギ・ヒラメの食いが急落する（魚が底に沈んで浮いてこない）
- `sst_delta_3d < 0 の連続日数` を SLOW 因子として追加することで、一時的な下落と継続的な低下を区別

**3. 大潮前後 2 日の「潮回り変化加速フラグ」**
- 「大潮前日〜大潮当日〜翌日」の潮力変化速度（tide_coeff の 1 日差分の絶対値）が高い日は、タチウオ・アジ系で食いが立つ
- `tide_coeff_delta = abs(tide_coeff_today - tide_coeff_yesterday)` として追加

### 既存因子の組合せ・非線形変換

**SST × 月齢の交互作用（タチウオ・内湾系）**
- 「SST が 20〜24℃ かつ大潮」のときのタチウオの爆釣は、単独因子の足し算では表現できない乗算的な影響
- `sst_x_tide = sst_avg × tide_coeff / 100` のような積特徴量
- ただし SLOW 因子同士の交互作用は過学習リスクがあるため MIN_N_COMBO の基準を高め（50件以上）て適用

**降水量 × 海域ゾーンの条件付き効果**
- 東京湾内湾では「雨後 2〜3 日で濁り → シーバス・シロギス急増」
- 外房・鹿島では「雨後の濁りは外洋潮流で数時間で洗われる」という正反対
- precip_sum3 は全エリアに同じ係数を当てているが、ゾーン別係数があるとエリア性格の差が出る

### 季節 transit の切り替わり

**現状の transition_risk の限界**
- 対象旬 ±2 の avg_cnt の変動係数（std/mean）→ 「ブレやすい旬」の識別には有効
- ただし「今年の切り替わりが例年より早い/遅い」を捉えられない

**追加すべき視点: SST × 季節 transit の組合せ**
- 春マダイ（4〜5月）の乗っ込み入りは水温で決まる。相模湾の SST が 15℃を超えた最初の週に乗っ込みが始まる
- spawn_season_n フラグは暦ベースだが、今年の SST が例年より低い（高い）ときは乗っ込み開始が 2〜3 週ズレる
- `sst_vs_climo`（当期 SST と過去同旬平均の差）という変数が必要

**秋シーバス（10〜11月）の切り替わり**
- 水温が 20℃を下回ると東京湾奥でシーバスが表層に浮く
- `sst_below_20_consec`（SST が 20℃を下回り続けた日数）が切り替わりシグナル

## まとめ: A/B/C/D のキー結論

**A 水色が効く魚種**: シロギス・シーバス・マダイが最上位。薄濁り(3)付近を最良とする中間適正型が多く、両端で釣果が落ちる**非線形（二次項）パターン**を現在の線形 OBS 因子では拾いきれていない可能性。深場・外洋回遊系（ヤリイカ・カツオ・キンメ）は水色無関係。

**B 空間補間の妥当性**: 東京湾内湾と外房の IDW 補間は海域性質が正反対なため NG。補間は**ゾーン内（AREA_TO_ZONE の既存定義）に限定**し、ゾーンまたぎはフォールバックに切り替える設計が現場感に合致。鹿島系は南沖（河川影響）と北沖・真沖（外洋系）でポイント別に補間元を分けるべき。

**C chowari 系船宿の特性**: point_place1 が薄く ship_fish_point.json の整備が予測精度の上限を規定する。**季節集中コンボに対する MIN_MONTHS 緩和（6→3）**が精度以前の参入障壁として存在する可能性。予測手法は分けなくてよいが、デフォルトポイント登録精度に先行投資が必要。

**D 未実装シグナル**: 風向転換（前日差分）・SST 連続低下フラグ・大潮前後の潮力変化速度 の 3 件。既存因子の組合せとしては SST × 月齢の積特徴量（タチウオ・内湾系向け）が効く可能性。季節 transit の改善は暦ベース→SST ベースのトリガー（`sst_vs_climo`、SST が閾値を下回った連続日数）への置き換えが本質的対処。
