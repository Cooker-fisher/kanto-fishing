🔧 実装仕様（crawler.py への追加）

1. 関数追加（テンプレート再利用のため独立化）
_ship_weekly_report_section(ship_name, area, today_dt, catches_7d, weather_7d) -> str
_ship_monthly_archive_section(ship_name, months_data: list[dict], today_dt) -> str

2. データ取得
・_load_ship_monthly_archive(ship_name, max_months=13) — data/V2/*.csv から月別集計
・_generate_weekly_report_text(ship_name, catches_7d, weather_7d) — テンプレート文章生成

3. 配置順（rev2）
HERO → 週次レポート → 基本情報BOX → 直近7日チャート → エリア内順位 → 料金プラン → 予約CTA → 📊 月別釣果実績（直近3ヶ月 + 折り畳み）→ FAQ → 座席設備

4. 「便数」表記の根拠
T29 commit 92b160804 で「件→便（同日複数便での重複カウント明示）」を確立済み・全サイトで統一

5. SEO 効果
・本文 ~2000字 → ~8000字（コンテンツ量 4倍）
・「{船宿名} {YYYY年MM月} 釣果」ロングテール 13個獲得
・新しい月が data/V2 に追加されると自動でカード追加（メンテ不要）
・週次レポートは毎週月曜 crawler 走行時に自動更新（鮮度シグナル）
