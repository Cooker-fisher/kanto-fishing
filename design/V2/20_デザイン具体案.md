## 2. 配色テーマ

### 方針: ライトテーマ + ダークネイビー&オレンジ（2026/04/04更新）

白背景ベースで清潔感・信頼感。**ダークネイビー+オレンジ**のアクセントで競合と差別化。
CSS変数で一元管理。

#### 競合との色差別化（2026/04/04 競合分析による）
| サイト | メインカラー |
|--------|------------|
| 船釣り.jp | グレー+水色+黄 |
| BoatAI | 水色ヘッダー |
| 釣りビジョン | 青 |
| **funatsuri-yoso** | **濃紺+白+オレンジ** ← 独自 |

→ 青系は競合と被るため回避。濃紺は信頼感、オレンジはCTA+釣りの朝焼けイメージ。

### 4.1 ZONE B': 予測の答え合わせ（新設）

**コンセプト**: 一昨日の予測結果から**一番成績の良かった魚種**を1件ピックアップ。
無料部分で「ちゃんと分析してる」をアピールし、有料部分（匹数・サイズ詳細）へ誘導。

#### チラ見せカード

```
┌──────────────────────────────────────────┐
│ 予測の答え合わせ（4/3の結果）             │
│                                          │
│ アジ ────────────────────── [的中]        │
│                                          │
│ 【前年比の予想 vs 実績】                  │
│  予想: 前年比+20%  →  実績: +25%         │
│                                          │
│ 【海況予報 vs 実績】                      │
│  水温  予報 18.2℃ → 実際 18.5℃  ○       │
│  風速  予報 5m    → 実際 4m     ○       │
│  波高  予報 1.2m  → 実際 1.0m   ○       │
│                                          │
│ 【影響分析】                              │
│  水温が適温帯に入り活性UP。              │
│  風速・波高とも予報通りで好条件が持続。   │
│┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄│
│ 【詳細予測 vs 実績】         [ブラー]     │
│  匹数  予想: 25〜35匹 → 実績: 20〜40匹   │
│  型    予想: 22〜30cm → 実績: 25〜38cm   │
│                                          │
│  → 全魚種の詳細予測を見る（月額500円）   │
└──────────────────────────────────────────┘
```

#### 無料/有料の線引き

| 項目 | 無料 | 有料 |
|------|------|------|
| 前年比（予想vs実績） | ○ | ○ |
| 海況予報vs実績の比較（○×） | ○ | ○ |
| 各要因の影響コメント | ○ | ○ |
| 匹数レンジ 予想vs実績（XX〜XX匹） | × ブラー | ○ |
| 型サイズ 予想vs実績（XX〜XXcm） | × ブラー | ○ |
| 重量 予想vs実績（XX〜XXkg） | × ブラー | ○ |
| 全魚種の答え合わせ | × 1件のみ | ○ 全件 |

※ 予想も実績も**レンジ表示**（XX〜XX匹 / XX〜XXcm）。人によって釣果が異なるため。

#### 有料予測の出力形式（D層実装時）
```
アジ（4/5 土）
  匹数: 25〜35匹
  型:   22〜30cm
  前年比: +20%
  海況: 水温18.2℃ / 風速5m / 波高1.2m / 大潮
  総合: ★★★★☆
```

#### ピックアップロジック
- 一昨日の全予測結果から**予想と実績の乖離が最小の魚種**を1件選出
- 的中していればアピール効果大
- 全部外れた日は表示しない or 「海況が急変しました」で誠実に報告

#### 原因帰属ロジック（D層実装時に組み込み）
1. 予測時に使った気象予報値（H=1〜7の forecast）を記録
2. 実際の気象実績値（weather_cache.sqlite）と比較
3. 各変数の予報vs実績を○×で表示
4. 差が大きい変数は影響コメントで説明
5. 「予報通りの値でモデルを再計算」→ 「予報通りなら+XX%」を算出可能
6. 海況が原因でない場合 → 素直にハズレ表示

#### 実装上の注意
- **D層（予測モデル）が未実装**のため、このゾーンはD層実装後に有効化
- HTMLテンプレート・CSSだけ先に作り、`<!-- ZONE B': D層実装後に有効化 -->` としてコメントアウト

```javascript
/**
 * showAnalysis(onComplete, cacheKey)
 *   cacheKey: sessionStorage のキー（同セッション2回目は短縮表示）
 *   onComplete: アニメ完了後のコールバック
 */
function showAnalysis(onComplete, cacheKey) {
  var overlay  = document.getElementById('analysis-overlay');
  var steps    = overlay.querySelectorAll('.step');
  var bar      = document.getElementById('analysis-progress-fill');
  var cached   = cacheKey && sessionStorage.getItem('spinner_' + cacheKey);

  overlay.classList.add('active');
  overlay.setAttribute('aria-hidden', 'false');

  if (cached) {
    /* 2回目以降: 0.5秒でバーだけ流してすぐ完了 */
    bar.style.width = '100%';
    setTimeout(function () { _finishOverlay(overlay, onComplete); }, 500);
    return;
  }

  /* ── D: ステッパー（初回）── */
  function activateStep(i, pct, delay) {
    setTimeout(function () {
      if (i > 0) {
        steps[i - 1].classList.remove('active');
        steps[i - 1].classList.add('done');
      }
      steps[i].classList.add('active');
      bar.style.width = pct + '%';
    }, delay);
  }

  activateStep(0, 35,  0);
  activateStep(1, 70,  1200);
  activateStep(2, 95,  2400);

  setTimeout(function () {
    steps[2].classList.remove('active');
    steps[2].classList.add('done');
    bar.style.width = '100%';
  }, 3200);

  setTimeout(function () {
    _finishOverlay(overlay, onComplete);
    if (cacheKey) sessionStorage.setItem('spinner_' + cacheKey, '1');
  }, 3600);
}

function _finishOverlay(overlay, onComplete) {
  overlay.classList.remove('active');
  overlay.setAttribute('aria-hidden', 'true');
  /* ステップをリセット（次回のために） */
  overlay.querySelectorAll('.step').forEach(function (s) {
    s.classList.remove('active', 'done');
  });
  var bar = document.getElementById('analysis-progress-fill');
  if (bar) bar.style.width = '0%';
  if (onComplete) onComplete();
}

/**
 * countUp(el, to, duration)
 *   el       : テキストを書き換える要素
 *   to       : 目標値（整数）
 *   duration : ms（推奨 800）
 */
function countUp(el, to, duration) {
  var start = null;
  var from  = 0;
  function step(ts) {
    if (!start) start = ts;
    var progress = Math.min((ts - start) / duration, 1);
    var eased    = 1 - Math.pow(1 - progress, 3); /* easeOutCubic */
    el.textContent = Math.round(from + (to - from) * eased);
    if (progress < 1) requestAnimationFrame(step);
  }
  requestAnimationFrame(step);
}

/**
 * showForecastResult(resultEl, counts, cacheKey)
 *   resultEl : .forecast-result 要素
 *   counts   : [{ el: DOM要素, to: 目標値 }, ...]
 *   cacheKey : showAnalysis と同じキー
 *
 * 使い方例（予測ボタンクリック時）:
 *   var btn = document.querySelector('.forecast-cta');
 *   btn.addEventListener('click', function () {
 *     var resultEl = document.querySelector('.forecast-result');
 *     var targets  = [
 *       { el: document.querySelector('.cnt-min'), to: 25 },
 *       { el: document.querySelector('.cnt-max'), to: 35 },
 *       { el: document.querySelector('.sz-min'),  to: 22 },
 *       { el: document.querySelector('.sz-max'),  to: 30 },
 *     ];
 *     showForecastResult(resultEl, targets, 'aji_20260412');
 *   });
 */
function showForecastResult(resultEl, counts, cacheKey) {
  showAnalysis(function () {
    /* F: 結果カード フェードイン */
    resultEl.classList.add('visible');
    /* F: カウンターアップ */
    counts.forEach(function (item) {
      countUp(item.el, item.to, 800);
    });
  }, cacheKey);
}
```
