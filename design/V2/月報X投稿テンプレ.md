# 月報 X 投稿テンプレ（2枚越し・船宿コメント付き）

月報（`/monthly/`）を X に投稿するための定型テンプレ。
**1枚目＝画像主役で結論／2枚目（1枚目へのリプライ）＝船宿への言及＋リンク** の2投稿構成。

---

## 運用ルール（必読・効果順）

1. **本文にURLを入れない** — X は外部リンク付き投稿の表示回数を絞る。リンクは必ず**2枚目（リプライ）**に置く。
2. **1枚目は画像を必ず添付** — `docs/x_post/` 系のカード PNG か、月報ベスト3を写した画像。数字（匹数・cm）が映えるものほど伸びる。
3. **2枚目で船宿に @メンション** — 新規〜小規模アカウントで最も効く拡散経路。船宿は地元の濃いフォロワーを持つため、いいね/RT されると一気に届く。ベスト3入りした船宿は「うちが1位」と拡散する動機がある。
4. **投稿タイミング** — 月初の**土日朝**（前月データが揃ったタイミング）。または金曜夜〜土曜早朝（釣行直前の情報収集層）。
5. **ハッシュタグは2〜3個** — 魚種・エリアタグ（例 `#タチウオ #東京湾 #船釣り`）。盛りすぎない。
6. 船宿の X ハンドルは事前に調べてメモ化（`@xxxx`）。不明な船宿はメンションせず**船宿名のみ**でも可（@なしでも言及自体に拡散効果あり）。

## テンプレ（プレースホルダ版）

### 1枚目（本文・画像添付・リンクなし）

```
【{YYYY}年{M}月 {魚種}月報】関東船釣り🎣

今月の{魚種}は{エリア/港}が好調でした。

📊月間ベスト3（数）
🥇{船宿1}（{港1}）{min1}〜{max1}{単位}／最大{size1}
🥈{船宿2}（{港2}）{min2}〜{max2}{単位}／最大{size2}
🥉{船宿3}（{港3}）{min3}〜{max3}{単位}／最大{size3}

詳しい日別データはリプ欄に👇

#{魚種} #{エリアタグ} #船釣り
```

### 2枚目（1枚目へのリプライ・船宿コメント＋リンク）

```
🥇の {船宿1}（{港1}）@{船宿1ハンドル} はこの日◯◯で絶好調。{ひとことコメント（竿頭の数・型・釣り方など現場感のある一文）}。

🥈🥉も {船宿2}・{船宿3} が安定。船宿さんお疲れ様でした🙇

▼{YYYY}年{M}月 {魚種}月報（全船宿・日別・エリア別）
{SITE_URL}/monthly/{romaji}.html
```

## 船宿コメント（2枚目）の書き方のコツ

- **kanso_raw（船長コメント）から現場感のある一文を1つ引く**。テンプレ感が消え、船宿が「ちゃんと読んでくれてる」と感じてRTしやすい。
  - 例：5/26タチウオ kanso_raw →「朝が快釣。am9:10でトップ29匹、トップ58匹はダントツ。100cm級連発」
- 数字（竿頭・最大型）を1つ入れると伸びる。
- 「お疲れ様でした」「アツいですね」など**船宿をねぎらう一言**を添えると拡散動機が上がる。
- 同一船宿が独占したときは「○○が独占」と素直に書く（事実なので船宿も喜ぶ）。
- 嘘・誇張はしない。数字は月報＝釣りビジョン配信データに準拠（min〜max は1人あたり最少〜竿頭）。

## 注意
- 「58匹」等は**船中合計ではなく竿頭（トップ1人）**の数。コメントで「トップ58匹」と書けば誤解を生まない。
- @メンションは船宿の公式アカウントが存在する場合のみ。なりすまし・非公式垢に当てない。
