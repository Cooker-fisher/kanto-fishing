# X 投稿フォーマット リサーチ（fish-value 計算結果シェア用）

**日付**: 2026-05-26
**目的**: `docs/fish-value/` の計算結果をワンタップで X に投稿できるフォーマットを設計
**ターゲット**: 35歳以上男性・釣り好き（サイトX解析実データ: 35-44歳 40.8% + 45-54歳 30.6% + 65+ 20.4% / Japan 91.2%）

---

## 1. 技術仕様の前提（Web Intent シェアURL）

X 公式の Tweet Intent URL は以下のパラメータが使える（Web Design Trends, ましじめ, ZIGZOW 各記事から確認）:

```
https://twitter.com/intent/tweet
  ?text=[本文]      ← URLエンコード必須
  &url=[URL]        ← 末尾に自動付加される
  &hashtags=tag1,tag2  ← カンマ区切り（# は自動付加）
  &via=funatsuri_yoso  ← 「@xxx より」と末尾に表示
  &related=funatsuri_yoso
```

**特殊文字**:
- スペース `%20`
- 改行 `%0A`
- `#` `%23`（hashtags パラメータ使用時は不要）

**URL長制限**: 文字数280字以内に収まればよい。URL は t.co 短縮で23字計算。

## 2. ゲーム界隈の高得点シェア定番パターン

具体的なテンプレ文面はゲーム公式 SDK の中に埋め込まれており Web 検索では引っかからない。ただし観測される定番パターンは以下:

### 2-A. スコア + ゲーム名 + ハッシュタグ型（多数）
```
スコア：999,999pt！自己ベスト更新！
#ゲーム名 #ハイスコア
[URL]
```
**特徴**: 数字を強調・自己ベスト/全国◯位/上位◯%等の文脈付与・絵文字最小限。

### 2-B. ランキング型（パズドラ・モンスト系）
```
🏆 ランキング◯位達成！
ステージ XX-X クリア
報酬：◯◯ゲット
[URL]
```
**特徴**: 達成感を絵文字で先頭強調・どこまで来たかを明示。

### 2-C. ガチャ・運要素強調型
```
SSR◯◯引きました！
天井◯回まで …でした 🎰
[URL]
```
**特徴**: 偶然・運の良さをネタにして共感を呼ぶ。釣果と相性◎（=釣りも運の要素大）

## 3. 釣り界隈のX投稿パターン（観測ベース）

### 3-A. 「みんなの釣果自慢」(@choka_jiman) などの拡散ハブ
釣果投稿を集約してRT する公式アカウント存在。RTされやすい投稿の特徴:
- **写真が主役**（クーラーボックス・船上・タモ）
- 本文は短い（80〜120字程度）
- 場所は曖昧にぼかす（県名 or 海域名どまり）
- 絵文字 1〜3個、ハッシュタグ 2〜4個

### 3-B. 釣り好きSNSで嫌われるパターン（カゴ釣り完全ガイド note より引用）
- 「地面のアスファルトや乾いた砂の上に、釣った魚をズラリと並べてる写真」=敬意欠如とされ炎上リスク
- 「背景に特徴的な堤防やテトラ、看板を映し込んで投稿」=情報漏洩・密漁ポイント化を誘発
- 数の競争（タチウオ50匹・尺アジ100匹）=食べきれるのか論争を呼ぶ
- 「『いいね』の数で価値が決まる」感を出すと逆効果

### 3-C. 推奨される要素
- **「その一匹に至るまでのプロセス」「ルアー選択と駆け引き」「海の様子・風の匂い」** などの体験記述
- 撮影は **水辺で・タモ網や濡れマット上で・素早く**
- ハッシュタグ平均 1.5倍インプレッション増（Gahara Blog）

### 3-D. 釣り絵文字定番
🎣🐟🐠🐡🦈🦐🐙 — 「絵文字のエモリスト」釣り絵文字まとめより

## 4. 金額換算・価値強調系の投稿事例

### 4-A. ふるさと納税系
「楽天お買い物マラソンで◯◯円分還元！」のような **数字をデカく**・**桁を強調**・**お得感** を出すフォーマットが定番。

### 4-B. 釣果の金額換算系（観測例）
釣り系ブログ・SNSで時折見かける:
- 「今日の釣果、スーパー価格にしたら ◯円相当」
- 「マダイ1キロ、市場なら3,500円」
- カゴ釣り愛好家・船釣り高齢層に「**ガソリン代の元取れたか**」議論あり → **金額換算は釣り中高年層に刺さるテーマ**

### 4-C. 換算アプリのシェア機能（カロリー換算・体重記録系）
- 「今日の歩数 = ◯◯km = 茶碗 ◯杯分」
- 「今月の睡眠 = ◯時間 = フライト◯往復」
- **「身近なものに換算」で意外性を出す → エンゲージ高い**

## 5. 35歳以上男性向け推奨要素

| 要素 | 推奨レベル | 根拠 |
|---|---|---|
| 絵文字 | 中（1〜3個） | おじさん構文（過剰絵文字）は逆効果。1〜3個が「自然」 |
| ハッシュタグ | 中（2〜4個） | 1.5倍インプレッション増・付けすぎると業者っぽい |
| 句読点 | 少なめ | 「。、」多用＝おじさん構文化リスク |
| カタカナ語尾 | NG | おじさん構文の象徴・若者に嘲笑される |
| 改行 | 多め | 読みやすさ重視・PCもスマホも改行は◎ |
| 写真添付 | 推奨 | 釣果投稿は写真ありが圧倒的に伸びる |
| 場所明示 | NG | 「神奈川」「相模湾」までに留める |
| 数字強調 | ◎ | 中年男性は数字・スペック大好き |
| 自己ベスト/記録 | ◎ | ゲーム高得点と同じく承認欲求の入口 |
| 自虐ネタ | 〇 | 「ボウズ回避」「ガソリン代の元取れず」等 共感 |

## 6. fish-value 用フォーマット提案 3案

URL生成は以下の Web Intent ベース:
```
https://twitter.com/intent/tweet?text={ENCODED_TEXT}&hashtags=funatsuri_yoso,釣果,船釣り
```

### 案A: 数字ドカン型（推奨・万人向け）

```
本日の釣果 ¥12,400相当 🎣

🐟 マダイ 1.2kg × 1匹
🐟 イサキ 0.4kg × 3匹
🐟 アジ 0.2kg × 5匹

計 9匹 / 3魚種

#船釣り予想 #釣果
```

**狙い**: 金額を1行目で先出し（=スコア型）・内訳は箇条書きで読みやすく・絵文字最小限

---

### 案B: ランキング・スコア型（ゲーム感）

```
🏆 今日の釣果スコア

💰 市場価格 ¥12,400相当
🐟 釣果 9匹（3魚種）

🥇 マダイ 1.2kg
🥈 イサキ 0.4kg × 3
🥉 アジ 0.2kg × 5

→ 自己ベスト更新／ガソリン代余裕で回収

#船釣り予想 #釣果スコア
```

**狙い**: スマホゲーム高得点投稿と同じ「達成感」演出・🥇🥈🥉 でランキング化・末尾に自虐 or 達成コメント

---

### 案C: シンプル換算型（カジュアル・ライト層）

```
今日の釣果、市場価格にしたら ¥12,400相当だった 🎣
（マダイ・イサキ・アジ 計9匹）

ガソリン代の元、取れたかな…

#船釣り予想 #釣果
```

**狙い**: 一文で完結・換算アプリのシェア機能と同じ気軽さ・「ガソリン代の元」フレーズで中年男性の共感狙う

## 7. 推奨運用フロー（programmer 向け実装メモ）

1. **デフォルト案A**（数字ドカン型）をワンタップ投稿のテンプレに採用
2. **オプションで案B / 案C を切替**できるラジオボタンを置くのは過剰 → **MVPは案Aのみ・後から拡張**
3. **写真添付は X 側で手動追加**（Web Intent は画像URL添付不可・ユーザーがツイート画面で追加）
4. **金額は ¥{wholesale_mid}相当** 表記で統一（卸値と小売値どちらにするかは UI 側で確定済の数字を使う）
5. **匹数 = 入力された総数**、**魚種数 = ユニーク魚種数** を計算して埋める
6. **URL は fish-value トップへ**（`https://funatsuri-yoso.com/fish-value/`）→ 流入経路として有効活用

### 文字数試算（案A）
```
本日の釣果 ¥12,400相当 🎣 (15字)
[改行2]
🐟 マダイ 1.2kg × 1匹 (13字)
🐟 イサキ 0.4kg × 3匹 (13字)
🐟 アジ 0.2kg × 5匹 (12字)
[改行]
計 9匹 / 3魚種 (9字)
[改行2]
#船釣り予想 #釣果 (10字)
[URL = 23字]
```
**合計 約120字** → 余裕で280字制限内・写真添付可能
