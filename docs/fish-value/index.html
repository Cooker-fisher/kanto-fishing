    算出方法

    価格データの根拠を表示

    卸売価格: 東京都中央卸売市場（豊洲）の月報明細を毎月取り込み、最新月の魚種別実勢単価（円/kg）を基準にしています。主要16魚種は日報（等級別の実取引価格）と突き合わせて検証し、乖離の大きい魚種は実勢に合わせて個別調整しています。

    季節補正: 蓄積した月報から魚種ごとの「暦月別の相場指数」を算出し、月報の公開ラグ（約1.5〜2か月）分の季節ズレをご利用月の相場水準に自動補正します（例: ブリは冬に高く初夏に安い）。

    サイズ換算: cm入力は魚種の体型別カーブで重量換算。同じ魚でもサイズ帯（小・標準・大・特大）で単価が変わる実態を、市場の等級差に基づく倍率で反映しています。

    小売換算: 卸売価格に魚種カテゴリ別の一般的な小売倍率（1.6〜3.0倍）を掛けた「丸ごと1尾」の概算です。柵・切り身など加工品の価格ではありません。

    ※ 本ツールは釣果の「おおよその市場価値」を楽しむためのものです。実際の取引価格・買取価格を保証するものではありません。
