# 日報検証レポート (2026/05/28)

## 概要

- データ源: 東京都中央卸売市場 日報（豊洲水産・販売結果）
- 取得日: 2026/05/12〜05/28 の 8 営業日
- 比較対象: `fish-price-master.json` v1 の wholesale_low / wholesale_high
- 目的: 月報単独項目（avg/high/low）から導出した推測値が、日々の実際の取引等級別単価とどの程度合っているかの検証

## 重要な前提

| 項目 | 月報 | 日報 |
|---|---|---|
| 単位 | 円/kg | 円/kg（同じ）|
| 期間 | 月間集計 | 日次（営業日）|
| 等級分け | 系統別品目（ブリ/ワラサ/イナダ/わかし）以外は混在 | 「銘柄」欄に「大/中/小・豆」等の等級表示が出現する |
| 出荷地 | 都道府県別 | 「主産地」または「各地」|
| 価格3値 | avg / high / low (全集計) | 高値 / 中値 / 安値（その日の取引内） |

日報の「高値・中値・安値」は同じ等級・同じ産地内での当日取引の幅。
月報の「low_overall / avg / high_overall」は月間全体（複数等級・複数産地混在）の幅。
