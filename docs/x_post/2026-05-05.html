1今日のハイライト

本日最大の話題はヒラメの27.00kg。5/5(火)時点で5月以降以降、関東圏での25kgkg超え大物記録です。小坪港は遠征エリアですが、洋征丸はこの時期の運航で近海では狙えないサイズの回遊魚を求める常連層に人気の船宿です。シーズン（過去5年同旬比）の 10.1倍という記録的な大物実績で、この時期ならではの好海況が背景にあると見られます。

全体感としては、今日は「アジ・マダイ・シロギス」のシーズン安定組と「ワラサ・イナダ・カンパチ」の単発大物・好機組に分かれる構成でした。 アジは平塚港・平和島エリアで3桁釣果が頻発し、群れの動きが継続中。巳之助丸の136匹は今週の全便でも上位の記録です。 マダイは大原港・鹿島港で0〜18匹レンジ。4.4kgの大型はサイズが明確に上向きな印象。 シロギスは平和島・東葛西の浅場で大爆発。湾奥への河川流入と水温上昇でベイト溜まりが形成された影響と推察されます。長崎屋の132匹は今期トップ級の記録です。 カワハギは久比里港・長井港で1〜20匹・5〜31cm。底物の繊細な釣りが好きな常連で常に予約が埋まる人気ジャンルで、本日も複数船宿で安定した釣果が記録されました。 タチウオは長浦・浦安方面で0〜26匹、サイズは60〜114cmと大型揃い。冬の名残のドラゴン級が春の早朝便で出ているパターンです。 カンパチは下田港で1〜3匹レンジ・2.0〜5.28kg。龍正丸が大物記録を出した遠征便で、近海では狙えないサイズ帯です。 マルイカは小網代港・松輪間口港で0〜52匹。たいぞう丸が継続的な好実績で、剣崎沖の良型群が継続中です。 ヤリイカは天津港・勝浦川津港方面で19〜49匹。不動丸など継続的に良型が記録されています。 スルメイカは松輪間口港・長井港方面で0〜18匹。ローテーション釣行に組み込みやすい魚種です。 全34魚種が顔を揃えた幅広い釣果構成。アジ・マダイ・シロギスが三本柱として件数を稼ぎ、マルイカ・タチウオ・ヒラメも少数ながら確実に記録されています。同日に複数魚種を狙えるため、釣行の自由度が高いタイミングと言えます。
