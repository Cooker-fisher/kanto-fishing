## Phase 3: 山本釣船店

| 項目 | 内容 |

| エリア | 清水港（巴川） |

| URL | kurodai.com |

| パーサー難度 | 高 |

- Type Bパーサーの開発は将来の他船宿対応にも活用可能

---

## Phase 4: 残り希薄層から効果の高い順

Phase 3完了後、02_カバレッジ分析.mdの「48.5%情報改善ポテンシャル層」から順次対応。

対応基準:
1. catches_raw件数が多い（影響範囲大）
2. 情報ロス率が高い（改善効果大）
3. HP構造が既存パーサーで対応可能（開発コスト低）
